# 🏦 Console Bank System (C++)

## 📌 About The Project
This project is a professional Console-based Bank Management System developed in C++. It is designed to demonstrate advanced proficiency in handling complex data structures, CRUD operations (Create, Read, Update, Delete), and file-based data persistence.

The system allows administrators to manage bank clients efficiently, providing a seamless and organized interface for all banking operations.

## 🚀 Key Features
- Client Management: Fully functional CRUD system for managing clients (Account Numbers, PINs, Names, Phones, and Balances).
- Financial Transactions: Built-in Deposit and Withdrawal modules with real-time balance validation and overdraft protection.
- Data Persistence: Automatically saves all changes to a bank.txt file, ensuring no data loss between sessions.
- Professional UI/UX: Clean, table-formatted console output using iomanip, with screen-clearing transitions for a smooth user experience.
- Error Handling: Robust validation to prevent duplicate account numbers and invalid transaction amounts.

## 🛠 Built With
* Language: C++
* Libraries: <iostream>, <vector>, <string>, <iomanip>, <cstdlib>, <fstream>

## 🧠 Technical Highlights
- Vector-based Storage: Utilizes std::vector for dynamic and efficient memory management.
- File Handling: Implemented file input/output streams to manage permanent storage of client records.
- Modular Architecture: Code is organized into clean, reusable functions, separating UI logic from data processing.
- User-Friendly Interface: Implemented a navigation system that ensures intuitive interaction.
